# My portfolio site

Hosted at [caseprince.com](http://caseprince.com)

## Setup

Install python 2 from https://www.python.org/downloads/release/python-2716/

> npm install -g bower

> bower install

> npm install

Install chocolatey: https://chocolatey.org/install

> choco install ruby

> gem install compass

> npm install -g grunt-cli

> grunt serve
